mchp-dev
========

Installation
------------

Clone the repository: 
$ git clone https://github.com/mitchellias/mchp-dev

Set up virtualenv: 
$ virtualenv3 ./mchp-dev

In the repository, install django:
pip install -r requirements.txt

I don't know if this is necessary yet:
